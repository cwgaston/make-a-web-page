HTML Markup

Plain text is plain text
Markup is made up of tags

Closing tag has / and brackets

em tag – emphasis – italics

HTML Attributes

Attributes always = value
Can have multiple attributes
Anchor tag = a in brackets and it's for making links
href=attribute name
Value =url
Contents – derp
images=img in brackets
Src =source
Alt = text if image doesn’t load
Images don’t have a closing tag – because it has no content

Whitespace

To make text on multiple lines add a break tag
This is a void tag – no need for closing tag
Paragraph tag can also be used and it is a block tag --which means there's an invisible block around it.

Inline vs Block

The break tag is inline and so is bold (b), italics (em), and image (img)
P tag is block … makes an invisibile box
A box that can be resized/shaped around the first p tagged text
Span is inline
Div is block

HTML Documents

In brackets with ! is the fact that it's an HTML doc
An opening tag and closing tag to surrounds the document html and then /html
Next is a head tag: meta data and other stuff, like if you want to use Javascript or CSS
Then a Title tag – which appears in your browser window as title
After title close your head tag
Then Body which is where we'll spend the bulk of time in this class
The tag here is "body"
And then don't forget to close the whole document with /html tag
